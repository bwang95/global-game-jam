    • Global Game Jam 2014: We

    Central Texas Global Game Jam, January 24th-26th, 2014

    Project maintained by bwang95 Hosted on GitHub Pages — Theme by mattgraham

    "We don't see things as they are, we see them as we are"

    We is a game designed around this theme. As one of three main characters, you the player will navigate a perilous dungeon, slaying monsters and collecting various items to aid you in your quest. Rather unfortunately, the lighting is poor...

    Game Objective

    Once you get all three cards, find the portal and get back out! You only have three lives, so be careful!

    Technical Information

    We is a roguelike, procedurally generated top-down dungeon crawler for Windows, made in Unity.

    Controls

    • Ctrl - Use Items
    • WASD - Movement
    • 1, 2, 3 - Change characters once you have unlocked them
    • Space - Attack; Each character has different attack and their respective enemies behave differently...Find the characters' cards to unlock them and find out what is different!

    Installation Instructions:

    Open global-game-jam/release/we.exe

    Troubleshooting:

    If the computer has a discrete graphics card (mainly those with Nvidia Optimus), the game may crash. To counteract this, set we.exe to run with the discrete graphics processor (for most Nvidia cards, right click on the exe and select Run with Graphics Processor -> Nvidia Processor).

    On Windows 8, the game may have to be run as administrator.
